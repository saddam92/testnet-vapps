### Description

EcoTrack helps individuals and businesses tokenize their carbon offset contributions as NFTs, creating a transparent, verifiable market for environmental impact. Users earn "Green NFTs" for sustainable actions (e.g., using public transport, renewable energy) which can be traded or displayed as proof of eco-consciousness.


### SL Integration  
- **Zero-Knowledge Proofs**: Verify carbon offset claims without exposing sensitive user data (e.g., income, location).
- **Immutable Ledger**: Store carbon credit transactions on SL for auditability.
- **Smart Contracts**: Automate NFT minting and trading rules.

## Technical

### Architecture

1. **Frontend**: Vue.js + Soundness SDK for ZK proof generation
2. **Backend**: Python (Flask) API for data processing
3. **Blockchain**: Soundness Layer (SL) for NFTs + Ethereum for cross-chain compatibility
4. **Storage**: IPFS for storing carbon project metadata

### Stack
- **Frontend**: Vue.js
